# Changelog

## v0.1.0 - 2026-07-12

Initial public release readiness for Hermes Intelligence Platform.

### Added

- Local-first decision intelligence runtime with SQLite memory.
- GitHub, paper, and domain RSS fixture-backed signal processing.
- Canonical `IntelligenceBrief` output contract and rule-owned decision actions.
- Zero-secret Markdown/Obsidian demo path.
- Static sample output under `examples/samples/`.
- First-run verification script.
- CI coverage for tests, compileall, smoke, acceptance, demo, and first-run checks.
- MIT license, security policy, issue templates, and PR template.

### Deferred

- Native dashboard UI.
- Vector DB and semantic entity linking.
- Semantic interest filtering from `HERMES_INTEREST_PROFILE` / `HERMES_RELEVANCE_THRESHOLD`.
- Specialized live APIs for Finance, Cybersecurity, Apple, NVIDIA, and Startup intelligence.
