# License Compliance

Intelligence Hub is distributed under the MIT License. See the repository root `LICENSE`.

## Contribution Rules

- Contributors must only submit code, documentation, fixtures, and images they are authorized to publish.
- Do not copy source code from projects with incompatible licenses.
- Record attribution and license requirements when incorporating third-party assets.
- Do not commit proprietary datasets, private documents, generated secrets, or user-owned Vault content.

## Dependencies

Runtime and development dependencies keep their original licenses. Before adding a dependency:

1. confirm its license permits the intended use and redistribution;
2. prefer maintained packages with clear provenance;
3. document any required notices;
4. avoid bundling external binaries or model weights without an explicit license review.

## Public Fixtures and Screenshots

Demo fixtures are repository-owned deterministic examples. Public screenshots must come from the zero-secret demo and must not expose private paths, credentials, IDs, or user data.
